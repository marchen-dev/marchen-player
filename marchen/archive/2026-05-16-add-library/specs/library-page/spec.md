## 目的

以卡片网格形式展示用户的影视库，支持筛选和排序。

### 需求: 卡片网格展示

系统 SHALL 以响应式卡片网格展示所有已入库的作品。

#### 场景: 正常展示

- **GIVEN** library 表中有多条作品记录
- **WHEN** 用户进入影视库页面
- **THEN** 以卡片网格形式展示所有作品
- **AND** 每张卡片显示：竖版海报、标题（单行截断）、观看进度（已看集数/总集数）、进度条
- **AND** 默认按最后观看时间降序排列

#### 场景: 空状态

- **GIVEN** library 表为空
- **WHEN** 用户进入影视库页面
- **THEN** 显示空状态提示（图标 + 文字说明）

#### 场景: 连载中标记

- **GIVEN** 某作品的 isOnAir 为 true
- **WHEN** 展示该作品卡片
- **THEN** 卡片上显示"连载中"标签

### 需求: 卡片交互

系统 SHALL 支持点击卡片打开详情面板，右键菜单提供快捷操作。

#### 场景: 点击卡片

- **GIVEN** 影视库列表中有作品卡片
- **WHEN** 用户点击某张卡片
- **THEN** 打开该作品的详情侧滑面板

#### 场景: 右键菜单

- **GIVEN** 影视库列表中有作品卡片
- **WHEN** 用户右键点击某张卡片
- **THEN** 显示上下文菜单，包含：查看详情、从库中移除
- **AND** 点击"从库中移除"后删除该 library 记录

### 需求: 筛选

系统 SHALL 支持按连载状态筛选作品。

#### 场景: 筛选连载中

- **GIVEN** library 中有连载中和已完结的作品
- **WHEN** 用户选择"连载中"筛选
- **THEN** 只显示 isOnAir 为 true 的作品

#### 场景: 筛选已完结

- **GIVEN** library 中有连载中和已完结的作品
- **WHEN** 用户选择"已完结"筛选
- **THEN** 只显示 isOnAir 为 false 的作品

### 需求: 排序

系统 SHALL 支持多种排序方式。

#### 场景: 按最近观看排序

- **GIVEN** 用户在影视库页面
- **WHEN** 选择"最近观看"排序
- **THEN** 按 lastWatchedAt 降序排列

#### 场景: 按评分排序

- **GIVEN** 用户在影视库页面
- **WHEN** 选择"评分"排序
- **THEN** 按 rating 降序排列

#### 场景: 按入库时间排序

- **GIVEN** 用户在影视库页面
- **WHEN** 选择"入库时间"排序
- **THEN** 按 addedAt 降序排列

### 需求: 路由和导航

系统 SHALL 在侧边栏提供影视库入口，替代原有的播放记录入口。

#### 场景: 侧边栏导航

- **GIVEN** 应用启动
- **WHEN** 用户查看侧边栏
- **THEN** 显示"影视库"导航项（替代原"播放记录"）
- **AND** 点击后导航到 /library 页面
