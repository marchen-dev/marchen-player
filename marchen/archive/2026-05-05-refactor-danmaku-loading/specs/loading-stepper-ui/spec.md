## 目的

展示弹幕加载进度的水平步骤指示器，让用户了解当前加载到哪一步。

### 需求: 步骤状态展示

系统 SHALL 以水平排列的方式展示 5 个步骤，每个步骤有三种视觉状态：已完成、进行中、未开始。

#### 场景: 步骤推进

- **GIVEN** 加载流程正在进行
- **WHEN** 当前步骤完成并进入下一步
- **THEN** 已完成的步骤显示完成标记（check icon）和高亮颜色
- **AND** 当前步骤显示进行中状态
- **AND** 未到达的步骤显示灰色

#### 场景: 步骤间连接线

- **GIVEN** 步骤之间有连接线
- **WHEN** 步骤已完成
- **THEN** 该步骤后的连接线显示高亮颜色
- **AND** 未完成步骤前的连接线显示灰色

### 需求: 纯 Tailwind 实现

组件 SHALL 仅使用 Tailwind CSS 类实现样式，不依赖任何 CSS 组件库。

#### 场景: 无外部样式依赖

- **GIVEN** 组件被渲染
- **WHEN** 检查组件的样式来源
- **THEN** 所有样式均来自 Tailwind 工具类和项目 CSS 变量（如 --primary、--muted）
